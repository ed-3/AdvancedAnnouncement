## config.yml

`config.yml` 中主要是一些基本的配置内容

```yaml
# 语言
  # zh_CN (简体中文) | en_US (English)
  translation: zh_CN
  
  # 是否打印公告到控制台，该选项只打印 CHAT 类型
  Console-broadCast: true
  
  # 启用 PlaceHolderAPI 支持
  PlaceholderAPI-support: true
  
  # 预公告的时间格式
  date-format: yyyy-MM-dd HH:mm:ss
  
  # 随机发送公告
  random: true
  
  # debug 测试
  debug: false
```
